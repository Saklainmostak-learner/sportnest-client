# SportNest

Premium sports facility booking platform built with MERN Stack.

## Live Links

Client:
https://sportnest-client-seven.vercel.app/

Server:
https://sportnest-server-h0si.onrender.com

## Features

- Firebase Authentication
- JWT Protected Routes
- Add / Update / Delete Facility
- Real Booking System
- Dashboard Overview
- Favorites & Cart
- Search & Filter
- Responsive Premium UI
- MongoDB Integration

## Tech Stack

### Frontend

- React
- Tailwind CSS
- Framer Motion
- React Router
- Firebase

### Backend

- Node.js
- Express.js
- MongoDB
- JWT
